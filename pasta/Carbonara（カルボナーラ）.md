## 食材
```
スパゲッティ(Spaghetti)：90g
卵黄：2個
ペコリーノ チーズ(Pecorino)：25〜30g（👉 卵黄の重量にほぼ近い量）
黒こしょう
肉（どれか1つ）
グアンチャーレ(guanciale)：40〜50g
（代用）パンチェッタ(pancetta)：50〜70g
（下位代用）ベーコン：60〜80g（脂が少ない）

クリーム不使用!
油不使用!

材料の準備：
卵黄を取り分ける
チーズを削っておく（できるだけ細かく）

```

## 作り方
1. 湯を沸かす
   ```
   湯に塩を入れる（入れすぎない）
   ```
2. 肉を焼く
   ```
   油不使用!
   冷たいフライパンに肉を入れる
   中火にかけ
   ```
3. パスタをゆでる
   ```
   表示時間より1分短くゆでる
   ```
4. ソースを準備する
   ```
   卵黄とチーズを混ぜる
   → paste, not cream
   黒こしょうを加える
   → not be shy with it
   ```
5. パスタをフライパンに移す
   ```
   パスタがゆで上がる1分前にフライパンを温め始める
   30秒ほど温めたら、ゆで汁をひとさじ加える
   パスタをフライパンに移す
   乾いている場合はゆで汁を少量ずつ足す
   乳化するまで加熱し、ソースと和える
   ⚠️65℃以上になると、卵が熱凝固する
   ```
6. 盛り付け
   ```
   仕上げにペコリーノチーズを振る
   ・ジューシーに仕上げたい場合は、ゆで汁を少量加えて調整する
   ```
