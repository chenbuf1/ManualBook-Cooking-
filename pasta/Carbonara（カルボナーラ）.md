## 食材
```
スパゲッティ(Spaghetti)：90g
卵黄：2個
ペコリーノ チーズ(Pecorino)：25〜30g（👉 卵黄の重量にほぼ近い量）

黒こしょう：適量
肉（どれか1つ）
グアンチャーレ(guanciale)：40〜50g
（代用）パンチェッタ(pancetta)：50〜70g
（下位代用）ベーコン：60〜80g（脂が少ない）

クリーム不使用!
油不使用!


```

## 作り方
1. 湯を沸かす
   ```
   湯に塩を入れる（入れすぎない）
   ```
2. 肉を焼く
   ```
   油不使用!
   冷たいフライパンに肉を入れる
   中火にかけ
   ```
3. 
 
 パスタをゆでる
```

沸騰したらパスタを入れる
表示時間より1分短くゆでる

```






 一定保留一杯面汤
Carbonara 的酱不是靠奶油，是靠面汤乳化

第二步：煎 Pancetta


不加油！



第三步：做酱（离火操作）

2 蛋黄 +  Pecorino 加入
大量黑胡椒
搅拌到 → 浓稠糊状（像湿沙子）

如果太干
👉 加 1 小勺热面汤
变成 浓厚膏体（不是液体）


第四步：真正的 Carbonara 乳化

面捞入锅中（带一点水）

开最小火 5 秒 → 关火

加入 2 勺面汤
快速搅拌 → 形成油水乳化

现在立刻：
👉 倒入蛋奶酱

持续快速翻拌

如果太稠
逐次加面汤

目标质地：

丝绸光泽、挂面、不流汤、不结块

温度控制：

65°C = 成功
75°C = 炒蛋

最后

装盘
再磨黑胡椒
少量 pecorino 雪花
