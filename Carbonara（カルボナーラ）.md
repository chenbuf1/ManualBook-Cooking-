
## 食材
```
白菜　200～250g
えのき　100g
豚バラ　200g

材料の準備：豚バラ肉は5cm程度に切る。白菜、えのきは食べやすいように切る。

白だし 25ml
水　250ml
ゆずポン酢

```

## 作り方
１、食材を入れ
   ```
   鍋に白菜、えのき、豚バラ肉を入れて白だしと水を注ぐ。

　　注意：白菜は芯を下、葉を上にする。吹きこぼれを防げる。
   ```

2、加熱
   ```
   中火にかけ、具材に火が通るまで煮る。
   ```
